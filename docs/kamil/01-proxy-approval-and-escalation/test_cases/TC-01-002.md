# TC-01-002: Dashboard Escalations tab displays escalated submissions

## Type
happy-path

## Preconditions
- At least 2 submissions are escalated (`is_escalated = true`, `status = 'pending'`)
- User is logged in as admin/platform_admin

## Steps
1. Navigate to the Dashboard page
2. Click on the "Escalations" tab

## Expected Result
- The Escalations tab shows a red badge with the correct count (2)
- Only escalated pending submissions are shown
- Each card has a red top border, warning icon, "Escalation" badge
- Each card shows "Escalation Approval Timeout" message with the REQ ID, stage info, and absent approver name
- Each card has "Reject" (red) and "Take Action" (orange) buttons
- Card shows vendor name, total amount, category, priority, reason, date, files count

## Priority
critical
