# TC-01-009: Non-admin users cannot see escalation actions

## Type
edge-case

## Preconditions
- An escalated submission exists
- User is logged in as a requester or approver (not admin)

## Steps
1. Navigate to the Dashboard
2. Check if the Escalations tab is visible
3. Try to access the escalated submission detail page

## Expected Result
- The Escalations tab is NOT visible to non-admin users (or shows 0 count)
- Non-admin users do not see escalation banners or Take Action buttons
- Escalation notifications are NOT sent to non-admin users
- Server actions for proxy approve and re-assign reject unauthorized users

## Priority
high
