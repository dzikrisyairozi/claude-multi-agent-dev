# TC-01-011: Translations work for all escalation UI

## Type
smoke

## Preconditions
- All escalation UI components are rendered
- Language switcher is available

## Steps
1. Navigate to Dashboard > Escalations tab
2. Switch language from English to Japanese
3. Open the Take Action dialog
4. Open the Proxy Approve dialog
5. Open the Re-assign Approver dialog

## Expected Result
- All UI labels, buttons, descriptions, and messages display in Japanese
- No English text remains visible in any dialog or card
- Layout does not break with Japanese text (which may be longer/shorter)

## Priority
medium
