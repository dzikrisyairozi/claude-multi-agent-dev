# TC-01-006: Reject escalated submission

## Type
happy-path

## Preconditions
- An escalated submission exists
- User is logged in as admin

## Steps
1. Navigate to Dashboard > Escalations tab
2. Click "Reject" on an escalated card
3. Enter rejection reason
4. Confirm rejection

## Expected Result
- Submission status changes to "rejected"
- `is_escalated` remains true (historical record)
- Requester receives a rejection notification
- Submission disappears from the Escalations tab
- Submission appears in the Rejected tab

## Priority
high
