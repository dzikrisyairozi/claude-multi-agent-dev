# TC-01-008: Escalation notifications appear in notification bell

## Type
happy-path

## Preconditions
- Cron has escalated at least one submission
- User is logged in as admin

## Steps
1. Click the notification bell in the header
2. Review the notifications list

## Expected Result
- "Submission Escalation" notifications appear with red warning icon
- Each shows "Purchasing request REQ-XXXX has expired due to no action within the due time"
- REQ ID is highlighted in blue/primary color
- "View Details" link navigates to the submission detail page
- Notification shows relative timestamp (e.g. "2 mins ago")
- Unread indicator (blue dot) appears on unread escalation notifications
- Notifications appear in both "All" and "Needs Action" tabs

## Priority
high
