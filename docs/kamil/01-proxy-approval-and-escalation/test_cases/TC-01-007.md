# TC-01-007: Submission detail page shows escalation banner

## Type
happy-path

## Preconditions
- An escalated submission exists
- User is logged in as admin

## Steps
1. Navigate to the submission detail page (e.g. /approval-requests/[id])

## Expected Result
- A red escalation banner is shown at the top with "Escalation" title and "Waiting for manager approval" subtitle
- "Reject" button (red) and "Take Action" button (orange/warning) are shown in the banner
- The rest of the detail page (Details, Files, Timeline tabs) renders normally below
- Clicking "Take Action" opens the same Take Action dialog as the dashboard

## Priority
high
